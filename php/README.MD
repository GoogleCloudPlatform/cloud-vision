# Google Cloud Vision API PHP examples

This directory contains [Cloud Vision API](https://cloud.google.com/vision/) PHP samples.


## Prereqisites
1. [Sign up for Google Cloud Platform Account](http://cloud.google.com)
2. [Enable Cloud Vision API](https://cloud.google.com/vision/docs/getting-started)
3. [Get App Engine SDK for PHP](https://cloud.google.com/appengine/downloads#Google_App_Engine_SDK_for_PHP)


## Setup
1. Alter creds.php with values that make sense to your project.
2. Create a project using GoogleAppEngineLauncher from SDK
3. Copy files to GoogleAppEngineLauncher project directory
4. Run
